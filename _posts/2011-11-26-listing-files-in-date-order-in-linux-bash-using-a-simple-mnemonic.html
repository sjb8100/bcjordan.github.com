Want to order a file listing by date modified? This is the command I’ve found myself using, mostly because it has a simple mnemonic of its arguments:

ls -last
  • -l – list in long format (one file per line, with file details)
  • -a – include files starting with .
  • -s – display filesize (in blocks). Incredibly unnecessary, because -l gives you number of bytes in file. But -last sounds cooler than -lat.
  • -t – sort by date / time modified, most recently modified first

Bonus: if you’d like to display the most recent files on the bottom of the list, throw in -r to reverse the listing order.

ls -lastr
